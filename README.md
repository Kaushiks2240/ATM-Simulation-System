🏦 Bharat Bank ATM Simulation System
A simple, interactive text-based Automated Teller Machine (ATM) simulation written in Python. This project demonstrates basic banking operations, user authentication, and secure balance management.

✨ Features
Secure PIN Verification: Users must enter the correct PIN (1234) to access their account, with a maximum of 3 attempts before lockout.
Balance Inquiry: View the current available account balance.
Cash Withdrawal: Withdraw money safely. The system handles insufficient balance errors to prevent overdrafts.
Cash Deposit: Add funds to the account securely.
Input Validation: Prevents crashes if a user enters invalid characters instead of numbers during transactions.
🚀 How to Run
Make sure you have Python installed on your computer.

Clone this repository or download the atm simulation.py file.

Open your terminal or command prompt.

Navigate to the folder containing the file.

Run the script using the following command:

python "atm simulation.py"
(Note: Use python3 if you are on a Mac or Linux machine).
